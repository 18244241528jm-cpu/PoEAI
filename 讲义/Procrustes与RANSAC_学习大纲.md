## 第 1 讲：Procrustes 问题与 Kabsch 算法

**对应讲义**：约第 7–14 页  

**目标**：理解“已知对应的 3D–3D 刚体配准”问题，掌握去中心、最大化 trace 与 Kabsch（SVD）闭式解。

| 内容 | 要点 |
|------|------|
| Procrustes 问题（刚体） | 两团 3D 点一一对应，求 $R, \mathbf{t}$ 使 $\mathbf{p}_i = R\mathbf{q}_i + \mathbf{t}$；已知对应，无外点。 |
| 为何 Homography 要 4 点、P3P 只要 3 点？ | 单应 8 dof（平面映射）；P3P 假设已标定，位姿 6 dof。 |
| Step 1：去中心 | 平移的最优估计：给定 $R$ 时 $\mathbf{t} = \bar{\mathbf{p}} - R\bar{\mathbf{q}}$；先求 $R$ 再求 $\mathbf{t}$。 |
| Step 2：代入 $\mathbf{t}$ 后最大化 trace | 最小化 $\sum \lVert \mathbf{p}_i - (R\mathbf{q}_i + \mathbf{t})\rVert ^2$ 化为最大化 $\operatorname{tr}(R^\top H)$，$H = \sum \tilde{\mathbf{p}}_i \tilde{\mathbf{q}}_i^\top$。 |
| Kabsch 算法 | $H = U\Sigma V^\top$ → $R = UV^\top$（或修正 $V$ 使 $\det R = 1$）；与 SVD/极分解的关系。 |
| Frobenius 范数 | 正交矩阵左乘不改变 F-范数；用于推导最优 $R$。 |
| 最少点数 | 不共线三点可唯一确定刚体；更多点用最小二乘。 |

**学完你能**：手推“去中心 → 最优 $R$ = 最大化 $\operatorname{tr}(R^\top H)$ → SVD 得 $R$”；实现或调用 Kabsch；与主线第 6 讲 P3P 后的 3D–3D 配准对应。

## 第 2 讲：外点与 RANSAC 思想

**对应讲义**：约第 15–26 页  

**目标**：理解为何需要外点拒绝，掌握 RANSAC 的基本流程与“最小子集 + 内点投票”思想。

| 内容 | 要点 |
|------|------|
| 外点问题 | 错误对应或误匹配会破坏 Procrustes / 单应 / PnP；最小二乘不鲁棒。 |
| 两类工具 | RANSAC（随机采样一致）、Hough 变换；本讲聚焦 RANSAC。 |
| RANSAC 核心 | 随机抽**最小子集** → 用最小子集**拟合模型** → **内点检验**（距离 < 阈值）→ 重复多次，选**内点最多**的模型。 |
| 最小子集大小 | 与模型**自由度（DoF）**对应：直线 2 点、单应 4 对、PnP 3 或 4 对等。 |
| 拟合模型 | 例如直线：用 2 点确定直线；单应：4 对点 DLT。 |
| 点到直线距离 | 一致的“距离”定义，用于判断内点；阈值需调节。 |
| 完整流程 | 采样 → 拟合 → 数内点 → 重复 $N$ 次 →（可选）用**全部内点**重新拟合。 |

**学完你能**：口述 RANSAC 的“采样 → 拟合 → 内点检验 → 选最佳”；说明最小子集与 DoF 的关系；写出点到直线距离用于内点判定。

## 第 3 讲：RANSAC 的迭代次数与参数

**对应讲义**：约第 27–30 页  

**目标**：会算“抽到一次全为内点的最小子集”所需试验次数 $N$，理解成功概率 $p$ 与内点率。

| 内容 | 要点 |
|------|------|
| 单次试验 | 随机抽最小子集；若子集中有外点，拟合的模型通常内点少。 |
| $N$ 次试验 | 做 $N$ 次独立试验，取内点最多的模型；$N$ 要足够大才能以高概率至少抽到一次“全内点”子集。 |
| 求 $N$ 的公式 | 设内点率为 $w$，最小子集大小为 $s$，要求以概率 $p$ 至少一次全为内点：$(1 - w^s)^N \le 1-p$ → $N \ge \log(1-p)/\log(1-w^s)$。 |
| 内点率 $w$ 未知时 | 用当前最大内点比例估计 $w$，动态更新 $N$（自适应 RANSAC）。 |
| 最后一步：Refit on inliers | 用 RANSAC 选出的**全部内点**重新拟合（如最小二乘直线、DLT 单应），得到更稳的最终模型。 |

**学完你能**：给定 $w, s, p$ 计算 $N$；解释为何要“用全部内点再拟合一次”；知道 $w$ 小或 $s$ 大时 $N$ 会很大（可解性）。

## 第 4 讲：RANSAC 应用与可解性

**对应讲义**：约第 31–32 页  

**目标**：把 RANSAC 套到直线拟合、单应估计上；理解“RANSAC 是否可解”与最小子集、内点率的关系。

| 内容 | 要点 |
|------|------|
| 直线拟合 | 最小子集 2 点 → 确定直线 → 点到直线距离判内点 → $N$ 次后 refit 于内点（最小二乘）。 |
| RANSAC for homography | 最小子集 4 对点 → DLT 求 $H$ → 重投影误差或对称转移误差判内点 → 选内点最多的 $H$ → 用全部内点再求 $H$。 |
| 与主线的衔接 | PnP：P3P + RANSAC（3 或 4 点最小子集）；平面：单应 + RANSAC（4 对点）；Procrustes：通常假设已剔除外点或与 RANSAC 配合。 |
| How tractable is RANSAC? | $N$ 随 $s$ 指数增长；内点率低时 $N$ 很大；可通过自适应 $N$、提前终止、多尺度等改善。 |

**学完你能**：对“直线”和“单应”各写出一套 RANSAC 步骤（最小子集、距离、$N$）；说明 RANSAC 在什么情况下计算量可接受、何时需改进策略。
