## 与前面的联系

| 前面的概念 | 在这里的复用 |
|-----------|------------|
| DLT / SVD | 八点法本质上也是 DLT；$E$ 分解用 SVD |
| RANSAC | 求 $E$ 时用 RANSAC 排除错误匹配（8 点 / 5 点 + RANSAC） |
| 单应 $H$ | 特殊情况：纯旋转或平面场景退化为单应 |
| 重投影误差 | 对极误差（点到对极线的距离）是类比概念 |
| $K$ 矩阵（内参） | $F = K^{-\top} E K^{-1}$，已标定时直接用 $E$ |

## 关键公式预览

1. **对极约束**：$\mathbf{x}_2^\top E\,\mathbf{x}_1 = 0$，其中 $E = [\mathbf{t}]_\times R$
2. **本质矩阵性质**：$\text{rank}(E) = 2$，两个相等奇异值
3. **基础矩阵**：$F = K_2^{-\top} E\, K_1^{-1}$
4. **视差与深度**：$Z = \dfrac{f \cdot B}{d}$
