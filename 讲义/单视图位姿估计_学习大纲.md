## 第 1 讲：几何与线性求解基础（Recap）

**对应讲义页码**：约 1–4 / 48  

**目标**：能清楚区分各类 2D/3D 变换，并会用 DLT 从点对应求变换。

| 内容 | 要点 |
|------|------|
| 射影变换 (Projective transformations) | 2D/3D 中仿射、射影的区别；自由度与不变量 |
| 直接线性变换 (DLT) | 从点对应写出线性方程组 \( Ax = 0 \)；超定时的最小二乘解 |
| 变换层次 (Hierarchy of transforms) | 欧式 → 相似 → 仿射 → 射影；何时用哪种模型 |

**学完你能**：给定 2D–2D 或 2D–3D 点对应，能写出 DLT 的矩阵形式并知道如何求解。

## 第 2 讲：平面单应 (Homography) 与位姿分解

**对应讲义页码**：约 5–10 / 48  

**目标**：理解“平面 + 单视图”下如何用单应矩阵估计相机位姿，并会从 H 分解出 R、t。

| 内容 | 要点 |
|------|------|
| 单视图下的相机位姿估计 | 问题设定：已知平面在像平面的投影，求相机相对该平面的位姿 |
| 平面诱导单应 (Plane-induced Homography) | 世界平面上的点 ↔ 图像点 的映射 \( H \)；\( H \) 与 \( K, R, t \)、平面法向的关系 |
| 从单应求位姿 | 将 \( H \) 分解为 (R, t)：列向量与旋转轴、平移方向的关系 |
| 得到合法旋转 | 为何 (R,t) 与 (-R,-t) 对应同一 \( H \)；如何选 det(R)=1 并做最近旋转校正 |
| 恢复平移尺度 | 利用“三列共享同一全局尺度”恢复 \( t \) 的尺度 |

**学完你能**：给定 \( H \) 和内参 \( K \)，写出或实现从 \( H \) 到 (R, t) 的分解步骤。

## 第 3 讲：April Tag 与平面位姿实战

**对应讲义页码**：约 11–18 / 48  

**目标**：把第 2 讲用于真实系统：用 April Tag 提供平面点对应，得到机器人相对平面的位姿，并做质量检查。

| 内容 | 要点 |
|------|------|
| April Tag 简介 | 什么是 April Tag；为何适合做平面上的已知 3D 点 |
| 机器人相对平面的位姿 | 问题：机器人相机相对“贴有 April Tag 的平面”的 (R, t) |
| 从 April Tag 到单应 | 世界坐标（Tag 角点）→ 像素坐标；用 DLT 或 OpenCV 求 \( H \) |
| 从单应恢复位姿 | 调用第 2 讲的分解得到 (R, t) |
| 结果不好的原因 | 非平面、误检测、运动模糊、标定误差 |
| 重投影检查 (Reprojection Check) | 用求得的位姿把 3D 点再投到图像，与检测点比较，用于筛选/调试 |

**学完你能**：在代码里用 April Tag 检测 → 单应 → 位姿 → 重投影，并解释常见失败原因。

## 第 4 讲：非平面场景与 PnP 问题

**对应讲义页码**：约 19–26 / 48  

**目标**：从“平面 + 单应”过渡到“任意 3D 点 + 单视图”的 PnP 问题，理解为何至少需要 3 个点。

| 内容 | 要点 |
|------|------|
| 非平面怎么办？ | 单应只适用于平面；一般场景需要 PnP |
| Perspective-N-Points (PnP) 问题定义 | 已知：N 个 3D 点（世界坐标）及其 2D 观测（像素）；求：相机外参 (R, t)（world-to-camera） |
| 方位角问题 (Bearing problem) | 灯塔类比：只知道方向、不知道距离时，两个灯塔无法定位置 |
| 为何至少需要 3 个点？ | 圆周角定理 (inscribed angle theorem)；第三个点消除歧义 |
| 3D 中的类似关系 | 从 2D bearing 到 3D 射线；图像平面上的像素 = 相机系下的一条射线 |
| PnP 的数学形式与 OpenCV | 写出 \( x \sim K[R\mid t]X \)；提及 `cv2.solvePnP` 等接口 |

**学完你能**：清楚区分 Homography（平面）和 PnP（一般 3D）；能说明“最少 3 个点”的直观与几何原因。

## 第 5 讲：PnP 的解法一 —— 非线性最小二乘

**对应讲义页码**：约 27–36 / 48  

**目标**：掌握用“最小化重投影误差”解 PnP 的思路，以及 Gauss-Newton、LM 和实现细节。

| 内容 | 要点 |
|------|------|
| 最小化重投影误差 | 残差 = 观测像素 − 用当前 (R,t) 投影的 3D 点；目标：最小化残差平方和 |
| 非线性最小二乘 vs 线性 | 线性最小二乘有解析解（正规方程）；PnP 的投影方程非线性，需迭代 |
| 线性化 | 在当前参数处泰勒展开，得到局部线性最小二乘 |
| 高斯-牛顿 (Gauss-Newton) | 迭代：线性化 → 解局部最小二乘 → 更新参数；雅可比 J：解析 vs 数值 |
| 列文伯格-马夸尔特 (Levenberg-Marquardt) | 步长控制；阻尼项；介于 GN 与梯度下降之间 |
| 实践中 | 写出残差函数，调用非线性优化库；解析雅可比收敛更快 |
| 注意事项 | 保持 R 为合法旋转（正交、det=1）；参数化用旋转向量或四元数 |
| 难点 | 外点 (outliers) 破坏结果；初值不好会导致不收敛 → 引出下一讲：鲁棒估计 |

**学完你能**：能写出重投影误差和残差形式，知道 GN/LM 在做什么，以及为何需要好的初值和鲁棒框架。

## 第 6 讲：PnP 的解法二 —— P3P 最小解与 RANSAC

**对应讲义页码**：约 37–48 / 48  

**目标**：理解“最少 3 个点可解 PnP但有歧义”，掌握 P3P 的几何与多项式求解思路，以及用第 4 点去歧义、用 RANSAC 抗外点。

| 内容 | 要点 |
|------|------|
| 最小解 + 外点拒绝 | 用最小点数（3 点）得到候选解，再用 RANSAC 投票，抗外点 |
| P3P 问题 | 3 个 3D 点 + 3 条射线；求相机位姿；最多 4 组解 |
| P3P Step 1：归一化到射线 | 用内参把像素转为单位射线方向 |
| P3P Step 2–3：三角与余弦定理 | 利用 3 点间距离与射线夹角建立方程；余弦定理 (Cosine Law) |
| 三角形余弦定理 | 得到关于 \( d_1, d_2, d_3 \)（相机到三点的距离）的方程 |
| P3P Step 4：消元求距离 | 从 (1)–(9) 式消元，得到关于一个变量的四次多项式；求根得距离 |
| 从距离到位姿 | 已知 3 个 3D 点及其在相机系下的 3D 位置 → 3D–3D 配准 |
| 3D–3D 配准 (Procrustes) | 已知两团 3D 点的对应，求 R、t；下次课会细讲 |
| 用第 4 个点去歧义 | 4 个点 ⇒ 唯一解；在 4 组 P3P 解中选重投影误差最小的 |
| 完整流程 | 最小子集（如 3 或 4 点）→ P3P 解 → RANSAC 选内点 → 可选：用全部内点做 bundle adjustment 或 solvePnP 精化 |

**学完你能**：能口述 P3P 的 4 步（射线 → 三角 → 余弦 → 四次多项式）；知道为何要第 4 点、为何要 RANSAC；能与第 5 讲的非线性方法对比（速度 vs 精度、鲁棒性）。

## 建议学习顺序与时间

- **第 1–2 讲**：连续学，打好几何和单应基础（约 1–2 次课）。
- **第 3 讲**：可配合小项目（如用 OpenCV + April Tag 跑通一例）。
- **第 4 讲**：承上启下，务必弄清“平面 vs 非平面”“单应 vs PnP”。
- **第 5–6 讲**：两种解法对照学；先 5（理解优化在做什么），再 6（理解为何需要 P3P+RANSAC）。

每讲后可做：  
- 用讲义对应页码复习；  
- 选 1–2 个公式自己推一遍；  
- 用 OpenCV/NumPy 实现一个小 demo（如 DLT、homography 分解、或 solvePnP 对比 P3P+RANSAC）。
