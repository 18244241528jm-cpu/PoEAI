## 补充 A-1：P3P 问题定义与 Grunert 解法

**对应文献**：Section 2（Problem Definition）, Section 3 中 Grunert’s Solution。

**目标**：与主线第 6 讲衔接，用文献记号严格写出问题并完整推一遍 Grunert 的四次多项式。

| 内容 | 要点 |
|------|------|
| 问题设定 | 已知三角形边长 $a,b,c$、单位射线 $\mathbf{j}_1,\mathbf{j}_2,\mathbf{j}_3$；未知距离 $s_1,s_2,s_3$；$p_i = s_i \mathbf{j}_i$。 |
| 余弦定理三式 | $s_2^2+s_3^2-2s_2 s_3\cos\alpha=a^2$ 等；(1)(2)(3) 的等价形式与符号约定。 |
| 变量替换 | $s_2 = u s_1,\ s_3 = v s_1$，消去 $s_1$ 得到关于 $u,v$ 的方程。 |
| Grunert 消元 | 从 (6)(7) 得到 $u$ 用 $v$ 表示，再代入得**四次多项式** (9)；系数 $A_4,\ldots,A_0$ 的表达式。 |
| 从根到距离 | 对每个实根 $v$ 得 $u$，再由 (4)(5) 得 $s_1,s_2,s_3$；最多 4 组“相机前方”的解。 |

**学完你能**：手推一遍 Grunert 从 (1)(2)(3) 到四次多项式的完整过程，并与主线第 6 讲的“Step 4”对应。

## 补充 A-2：六种 P3P 解法的代数比较

**对应文献**：Section 3 中 “Outline of the Differences”, Finsterwalder, Merritt 等解法；Figure 2 流程图。

**目标**：理解不同解法在“变量替换、选哪两对方程、如何消元”上的差异，不要求每类都手算。

| 内容 | 要点 |
|------|------|
| 六种解法来源 | Grunert (1841), Finsterwalder (1937), Merritt (1949), Fischler & Bolles (1981), Linnainmaa et al., Grafarend et al. |
| 变量替换差异 | 多数用 $s_2=u s_1,\, s_3=v s_1$；Linnainmaa 等用 $s_2 = u + \cos\gamma\, s_1$ 等形式。 |
| 方程对的选择 | 用 (1)(2) 与 (2)(3)、或 (1)(2) 与 (1)(3) 等不同组合消元。 |
| 消元策略 | 代入消元（Grunert, Merritt）vs 直接消元（F&B, Linnainmaa）vs 引入新变量 $\lambda$（Finsterwalder, Grafarend）。 |
| Finsterwalder 特点 | 引入 $\lambda$ 使某判别式为完全平方，得到**三次**多项式 + 两次二次，而非单一四次。 |

**学完你能**：能对照 Figure 2 说明六种解法在“变量替换 → 方程对 → 消元 → 新变量”上的区别，并知道为何会得到四次或“三次+二次”等形式。

|------|------|

## 补充 B-1：SVD 的定义与基本性质

**目标**：掌握 $A=USV^\top$ 的构成、与 $A^\top A$ / $AA^\top$ 的关系、四子空间与 $A\mathbf{v}_j = \sigma_j \mathbf{u}_j$。

| 内容 | 要点 |
|------|------|
| SVD 陈述 | 任意 $m\times n$ 的 $A$ 可写为 $A = USV^\top$：$U$ 正交、$V$ 正交、$S$ 对角（非负奇异值 $\sigma_1,\ldots,\sigma_r$）。 |
| 与特征值的关系 | $U$ 的列是 $AA^\top$ 的单位特征向量；$V$ 的列是 $A^\top A$ 的单位特征向量；$S$ 对角线为 $A^\top A$（或 $AA^\top$）非零特征值的**平方根**。 |
| 四子空间 | $U$ 前 $r$ 列 → 列空间；$U$ 后 $m-r$ 列 → 左零空间；$V$ 前 $r$ 列 → 行空间；$V$ 后 $n-r$ 列 → 零空间。 |
| 关键关系 | $A\mathbf{v}_j = \sigma_j \mathbf{u}_j$；$A^\top A \mathbf{v}_j = \sigma_j^2 \mathbf{v}_j$；从 $A^\top A \mathbf{v}_j = \sigma_j^2 \mathbf{v}_j$ 推出 $A\mathbf{v}_j$ 是 $AA^\top$ 的特征向量。 |
| 数值与条件数 | 正交矩阵不改变向量长度；条件数 $\sigma_{\max}/\sigma_{\min}$（可逆时）；小奇异值导致不稳定。 |

**学完你能**：对给定小矩阵手算或编程求 SVD；能解释 DLT 中“取 $V$ 最后一列”对应最小奇异值。

---

## 补充 B-2：SVD 的应用（伪逆、最小二乘、极分解）

**目标**：会用 SVD 理解最小范数最小二乘解、伪逆 $A^+$、以及极分解在“旋转+伸缩”中的含义。

| 内容 | 要点 |
|------|------|
| 秩 1 分解 | $A = \sigma_1 \mathbf{u}_1 \mathbf{v}_1^\top + \cdots + \sigma_r \mathbf{u}_r \mathbf{v}_r^\top$；图像压缩中保留前 $k$ 项。 |
| 有效秩 | 用奇异值大小与阈值判断“有效秩”，比数主元更稳。 |
| 极分解 | 任意方阵 $A = QS$，$Q$ 正交、$S$ 对称半正定；$S = V\Sigma V^\top$，$Q = UV^\top$；连续介质/机器人中的“旋转+伸缩”解释。 |
| 最小二乘与伪逆 | $Ax=b$ 无解或解不唯一时，**最小范数最小二乘解** $x^+ = A^+ b$；$A^+ = V S^+ U^\top$，$S^+$ 对角元为 $1/\sigma_i$（$\sigma_i>0$）。 |
| 与主线联系 | DLT 齐次方程 $\min\lVert Ah\rVert ^2$ 取 $V$ 最后一列；Procrustes 中 $R=UV^\top$；条件数在求解稳定性中的角色。 |

**学完你能**：用 SVD 写出 $x^+ = A^+ b$ 并说明为何是最短解；能说明极分解 $A=QS$ 在几何上的意义；与主线中 DLT、Procrustes 的用法对应起来。
